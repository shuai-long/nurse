- 三无: 无责任性损坏、无药品过期变质、无器械破损性能失灵
- 三及时: 及时检查、及时补充、及时维修
- 四定: 定数量、定位置、定基数卡片、定期消毒灭菌
- 停电:
  1. 停电通知后，做好停电准备，备好应急灯、手电，呼吸机患者备好简易呼吸器
  2. 突然停电，立即启动抢救患者机器运转的替代方法，维持抢救工作并开启应急灯，联系水电维修班，向有关领导汇报
  3. 维持病房秩序，组织人力保证患者医疗安全
  4. 加强巡视，安抚患者，并及时解决患者问题，防火，防盗
